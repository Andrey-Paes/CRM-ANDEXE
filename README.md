# CRM
Sistema de gestão de clientes Full-Stack 
Em Desenvolvimento!

O Andexe-CRM é uma plataforma de gestão de relacionamento com clientes desenvolvida para organizar, automatizar e otimizar processos comerciais.

O sistema permite o gerenciamento completo de leads, controle de funil de vendas, acompanhamento de oportunidades e análise de desempenho através de dashboards estratégicos.

Projetado com foco em simplicidade, segurança e escalabilidade, o Andexe CRM integra landing pages, banco de dados em tempo real e controle de acesso por níveis de usuário, garantindo organização, produtividade e tomada de decisão baseada em dados.

O objetivo do projeto é oferecer uma solução moderna, intuitiva e eficiente para empresas que desejam profissionalizar sua gestão comercial.
